# 老黄历日历

一个简洁美观的老黄历日历应用，支持每日宜忌查询和事件管理。

## 功能

- 📅 日历视图，支持月切换
- 🧧 每日宜忌、冲煞、吉神显示
- 📱 响应式设计，兼容电脑和手机
- 💾 事件本地存储（LocalStorage）
- 🎋 传统风格界面

## 技术栈

- 原生 HTML/CSS/JavaScript
- [lunar-javascript](https://github.com/6tail/lunar-javascript) - 黄历计算库

## 快速开始

```bash
# 直接用浏览器打开
open index.html

# 或者启动本地服务器
npx serve .
```

## 访问

- 本地: http://localhost:8001
- 在线: 部署后访问公网地址:8001
